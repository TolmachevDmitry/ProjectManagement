По итогам **Фазы 1** был выбран проект - **Игра-стратегия**.

## UC-01 — Постройка новой базы

| Поле                        | Описание                                                                    |
| --------------------------- | --------------------------------------------------------------------------- |
| **Акторы**                  | Игрок                                                                       |
| **Предусловия**             | У игрока есть доступная территория и ресурсы                                |
| **Постусловия**             | База построена или действие отменено                                        |
| **Позитивный сценарий**     | Игрок выбирает тип здания → место → подтверждает → строительство начинается |
| **Альтернативный сценарий** | Игрок меняет место или выбирает другое здание                               |
| **Негативный сценарий**     | Недостаточно ресурсов или зона занята                                       |
